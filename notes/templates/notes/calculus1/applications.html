1. Critical Numbers & Extrema

To find the highest or lowest points on a graph, we must first identify the "candidates." Calculus tells us that peaks and valleys can only occur at specific locations called Critical Numbers.

Definition: Critical Number

A value \(c\) in the domain of \(f\) is a Critical Number if:

$$f'(c) = 0 \quad \text{(Flat Tangent)}$$
$$\text{OR}$$
$$f'(c) \text{ is Undefined} \quad \text{(Sharp Corner/Cusp)}$$
Types of Extrema
  • Absolute Max/Min: The highest/lowest point on the entire domain.
  • Local Max/Min: The highest/lowest point relative to the immediate surroundings (the top of a hill).

2. Existence Theorems

These theorems guarantee that a specific point exists without telling you exactly how to find it.

Rolle's Theorem

If you start and end at the same height (\(f(a)=f(b)\)), you must have turned around somewhere (\(f'(c)=0\)).

Mean Value Theorem (MVT)

At some point, your instantaneous speed (derivative) must equal your average speed.

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

3. Derivative Tests: Shape & Curvature

We analyze the derivatives to determine the behavior of the function.

First Derivative Test (Direction)
  • \(f' > 0 \rightarrow\) Increasing (\(\nearrow\))
  • \(f' < 0 \rightarrow\) Decreasing (\(\searrow\))

Use to classify Critical Points: \(+ \to -\) is a Max; \(- \to +\) is a Min.

Second Derivative Test (Concavity)
  • \(f'' > 0 \rightarrow\) Concave Up (\(\cup\))
  • \(f'' < 0 \rightarrow\) Concave Down (\(\cap\))

Inflection Point: A point where concavity changes sign (requires \(f''=0\)).

4. L'Hôpital's Rule

Limits involving infinity or zero often get stuck. L'Hôpital's Rule lets us take the derivative of the numerator and denominator to unblock the limit.

The Golden Rule

You can ONLY apply the rule if the limit is a fraction \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\).

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

Type 1: Basic Quotient (\(\frac{0}{0}\) or \(\frac{\infty}{\infty}\))

Problem: \(\lim_{x \to 0} \frac{\sin x}{x}\)   (\(\frac{0}{0}\))

Solution: Derive top and bottom.

$$\lim_{x \to 0} \frac{\cos x}{1} = \frac{1}{1} = \mathbf{1}$$

Type 2: The Product (\(0 \cdot \infty\))

Problem: \(\lim_{x \to 0^+} x \ln x\)   (\(0 \cdot -\infty\))

Solution: Rewrite multiplication as division by the reciprocal.

$$\lim_{x \to 0^+} \frac{\ln x}{x^{-1}} \quad \left(\text{Now } \frac{-\infty}{\infty}\right)$$

Apply L'Hôpital:

$$\lim_{x \to 0^+} \frac{1/x}{-x^{-2}} = \lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = \mathbf{0}$$

Type 3: The Difference (\(\infty - \infty\))

Problem: \(\lim_{x \to 0} (\csc x - \cot x)\)   (\(\infty - \infty\))

Solution: Find a common denominator to combine them.

$$\lim_{x \to 0} \left(\frac{1}{\sin x} - \frac{\cos x}{\sin x}\right) = \lim_{x \to 0} \frac{1 - \cos x}{\sin x} \quad \left(\text{Now } \frac{0}{0}\right)$$

Apply L'Hôpital:

$$\lim_{x \to 0} \frac{\sin x}{\cos x} = \frac{0}{1} = \mathbf{0}$$

Type 4: The Power (\(0^0, 1^\infty, \infty^0\))

Problem: \(y = \lim_{x \to \infty} x^{1/x}\)   (\(\infty^0\))

Solution: Use \(\ln\) to bring the power down.

$$\ln y = \lim_{x \to \infty} \frac{1}{x} \ln x = \lim_{x \to \infty} \frac{\ln x}{x} \quad \left(\frac{\infty}{\infty}\right)$$

Apply L'Hôpital to finding \(\ln y\):

$$\lim_{x \to \infty} \frac{1/x}{1} = 0 \quad \rightarrow \quad \ln y = 0$$

Final Step (Undo Log): \(y = e^0 = \mathbf{1}\)

5. Antiderivatives

Up until now, we have been doing differentiation: given \(f(x)\), find the slope \(f'(x)\). Antidifferentiation is the reverse process: given the slope \(f(x)\), reconstruct the original function \(F(x)\).

Definition

A function \(F\) is called an antiderivative of \(f\) if:

$$F'(x) = f(x)$$
The Most General Antiderivative

If \(F(x)\) is an antiderivative of \(f(x)\), then the most general antiderivative is:

$$F(x) + C$$

Why "+ C"? The derivative of any constant is zero. \(x^2\), \(x^2 + 10\), and \(x^2 - 5\) all have the same derivative (\(2x\)). \(C\) represents this unknown vertical shift.

Basic Reverse Rules

Given \(f(x)\) Antiderivative \(F(x)\)
\(x^n\) \((n \neq -1)\) \(\frac{x^{n+1}}{n+1}\) (Add 1 to power, divide by new power)
\(k\) (Constant) \(kx\)
\(\cos x\) \(\sin x\)
\(\sin x\) \(-\cos x\) (Note the negative!)
\(\sec^2 x\) \(\tan x\)

Example: Finding the Specific Function

Problem: Find \(f(x)\) if \(f'(x) = 2x\) and \(f(0) = 4\).

1. General Antiderivative: \(f(x) = x^2 + C\)

2. Use the condition \(f(0) = 4\) to find \(C\):

$$(0)^2 + C = 4 \quad \rightarrow \quad C = 4$$

Answer: \(f(x) = x^2 + 4\)
